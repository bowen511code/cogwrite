# PRD（产品需求文档）— CogWrite

## 1. 背景与目标
- 目标：做一个“认知科学 × AI”的自媒体写作助手，输入选题/资料，输出提纲、金句、引用、互动问题。
- 产出形式：Markdown 草稿 + JSON（可用于后续编辑/发布）。

## 2. 用户与使用场景
- 用户：做认知科学/心理学/教育学相关内容的个人创作者。
- 场景：快速从资料整理出一篇可发布的结构化草稿。

## 3. 核心功能（MVP）
- 生成提纲（outline）
- 生成金句（punchlines）
- 生成互动问题（interactive_questions）
- 生成引用（citations，必须可追溯；证据不足需标注）

## 4. 非功能需求
- 输出结构稳定：可解析、字段齐全（结构化 JSON）。
- 引用可靠：引用必须附带证据片段与来源标识。
- 可复现：同一输入在合理范围内输出风格一致。
